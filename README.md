# Docker Jenkins file

This is the Docker Jenkins image for the Docker Fundamentals capstone
exercise.

Run: `docker run -it -v /var/run/docker.sock:/var/run/docker.sock <imagename>`

The jobs are pre-populated and should work out of the box.
